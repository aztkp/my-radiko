## ログの追加方法

1. Chrome拡張で番組を聴きながらメモを入力
2. 「保存」ボタンでローカルに保存
3. 「履歴」ボタンでMarkdownエクスポート
4. エクスポートしたファイルを `logs/YYYY/MM/DD.md` に配置
5. このREADMEのカレンダーにリンクを追加
